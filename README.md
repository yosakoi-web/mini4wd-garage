# MINI 4WD SETTING GARAGE

GitHub Pagesだけで動くミニ四駆セッティング記録アプリです。ChatGPTログインや外部APIは使いません。

## GitHub Pagesで公開する方法

1. このフォルダ内のファイルをリポジトリの一番上へアップロードします
2. GitHubの `Settings` を開きます
3. 左側の `Pages` を開きます
4. `Source` を `Deploy from a branch` にします
5. Branchを `main`、フォルダを `/ (root)` にして `Save` を押します
6. 数分後に `https://yosakoi-web.github.io/mini4wd-garage/` を開きます

GitHubのリポジトリ画面にREADMEが表示されるのは正常です。アプリ本体はGitHub PagesのURLから開きます。

## 保存について

マシン情報と写真は、各端末のブラウザ内に保存されます。複数台登録できます。ただし、別のスマホやパソコンとは自動同期されません。ブラウザのサイトデータを消すと登録内容も消えます。

4人で共有するチャットと写真同期には、GitHub Pagesとは別にデータベースが必要です。現版にはチャットを入れていません。

## 更新履歴

バージョン2では、古い登録データがある場合にマシン名の保存後画面が空欄へ戻り、写真がないのに写真容量のエラーが表示される問題を修正しました。保存失敗時は入力内容を残します。

バージョン3では、スマートフォンの決定操作を次の入力欄へ進む動作に変更しました。保存は画面下部の「ガレージへ保存」ボタンでのみ実行します。登録カードからマシン詳細を開き、写真、基本セッティング、全パーツ、メモを確認できます。

バージョン4では、モーターの長所、注意点、向くコース、ギア比の目安を確認できるモーターガイドを追加しました。登録済みマシンはカードまたは詳細画面から編集でき、同じマシンへ変更を上書き保存できます。

バージョン5では、総重量を入力すると100g基準の重量補正速度、変化率、変化する速度を表示します。重量の影響は路面やモーター特性で変わるため、100gを超える場合は10g増加につき約1.8%低下する簡易目安です。登録後の詳細画面でも重量補正速度を確認できます。

バージョン6では、現在のセッティングとは別に走行・完走セッティング履歴を保存できます。走行日、コース名、結果、タイム、走行メモと、その時点の写真、基本設定、全パーツ構成を固定保存します。過去の構成を編集画面へ呼び出して、そこから次のセッティングを作れます。
